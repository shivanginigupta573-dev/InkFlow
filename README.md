# ❄️ InkFlow

**InkFlow** is a modern, minimalist blogging and creative writing platform. It is designed with a pleasing aesthetic—using soft baby-blue palettes and fluid transitions to create a calm, focused environment for writers.

## 🎨 UI Features
* **Frozen Waves**: Custom-built layered SVG wave transitions at the hero section.
* **Minimalist Design**: A clean, airy layout using the *Plus Jakarta Sans* typeface.
* **Empty State Logic**: A user-friendly "No writings here yet" display for new accounts.

## 🛠️ Tech Stack
* **Framework**: Django
* **Styling**: Tailwind CSS
* **Database**: SQLite
* **Icons**: Open Doodles & Popsy
